# Shadow

Depth determines the position level of each component and content depending on
page's hierarchy and focus. Shadows are used to express depth of elements from
each other.

```TypeScript
import { Shadow } from 'glints-aries';

// In CSS
const MyCard = styled.div`
  box-shadow: ${Shadow.down1};
`;

// With Box
<Box boxShadow={Shadow.down1} />
```
